# Hero experience wallpaper images

Place the rotating desktop hero images here. The homepage uses these paths:

- `experience-camping.jpg`
- `experience-party.jpg`
- `experience-diy.jpg`
- `experience-photo.jpg`
- `experience-movie.jpg`

Use landscape, lifestyle/experience imagery (camping, party, DIY, photography, movie). Images are shown only on desktop (md+), fade left-to-right, and rotate every ~10 seconds with a crossfade.
