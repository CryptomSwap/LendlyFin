# Hero experience wallpaper images

Place the rotating desktop hero images here. The homepage uses these paths:

- `experience-camping.jpg`
- `experience-party.jpg` (or `party.jpg`)
- `experience-diy.jpg`
- `gardning.jpg` (or similar)

Use landscape, lifestyle/experience imagery (camping, party, DIY). Images are shown only on desktop (md+), fade left-to-right, and rotate every ~20 seconds with a crossfade.
